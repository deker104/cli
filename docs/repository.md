# Организация репозитория

## Структура проекта
```
cli-interpreter/
│── docs/
│   ├── architecture.md         # Основной документ
│   ├── components.md           # Подробное описание компонентов
│   ├── flow.md                 # Взаимодействие компонентов
│   ├── repository.md           # Описание структуры репозитория
│   ├── checklist.md            # Чеклист
│── cmd/                        # Точка входа
│   ├── main.go                 # Основной файл CLI
│── internal/                   # Внутренние модули
│   ├── parser/                 # Разбор командной строки
│   │   ├── parser.go           # Код парсера
│   │   ├── parser_test.go      # Тесты парсера
│   ├── executor/               # Исполнение команд
│   │   ├── executor.go         # Код исполнителя
│   │   ├── executor_test.go    # Тесты исполнителя
│   ├── env/                    # Управление переменными окружения
│   │   ├── env.go              # Код переменных окружения
│   │   ├── env_test.go         # Тесты переменных окружения
│   ├── pipes/                  # Обработка пайпов
│   │   ├── pipes.go            # Код пайпов
│   │   ├── pipes_test.go       # Тесты пайпов
│── tests/                      # Дополнительные интеграционные тесты
│── .github/workflows/          # Настройка CI
│── docs/                       # Документация
│── go.mod                      # Go-модуль
│── go.sum                      # Go-зависимости
│── README.md                   # Описание сборки и запуска
```
